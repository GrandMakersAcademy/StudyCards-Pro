# StudyCards-Pro

Advanced flashcard learning system with spaced repetition algorithm for effective long-term memorization. Perfect for university students, language learners, and anyone looking to master complex subjects efficiently.

## Features

- Intelligent spaced repetition (SM-2 SuperMemo algorithm)
- Progress tracking and statistics
- Multiple categories and decks (for math, history, etc.)
- Card editor with examples and tags
- Modern PySide6 GUI with qdarkstyle dark theme
- Import/export decks (CSV, JSON)
- Detailed English documentation

### Installation & Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/GrandMakersAcademy/StudyCards-Pro.git
   cd StudyCards-Pro
   ```
2. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch the app:
   ```bash
   python main.py
   ```

## For Education
This program is designed for effective learning and memorization, ideal for institute and university students, science, languages, and exam preparation. All data stored locally in SQLite.
